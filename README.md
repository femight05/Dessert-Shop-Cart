# 🍰 Dessert Cart

A responsive dessert shopping cart application built with React. Users can browse a selection of desserts, add and remove items from their cart, update quantities, view the total order price, and confirm their order through a clean confirmation modal.

## 🚀 Live Demo

- **Live Site:** Not yet deployed...still working on the bugs
- **Repository:** **https://github.com/femight05/Dessert-Shop-Cart**

## ✨ Features

- Browse a collection of desserts
- Add products to the cart
- Increase and decrease product quantities
- Remove items from the cart
- Automatically calculate the total order price
- Display an order confirmation modal
- Start a new order after confirmation
- Responsive design for mobile, tablet, and desktop devices

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/femight05/Dessert-Shop-Cart
```

Navigate into the project:

```bash
cd Dessert-Shop-Cart
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
